# Terms Template

Short glossary entries. One term per file, or multiple terms in one file if they form a cluster.

---

```markdown
# <Термин>

**Определение:** Одно-два предложения. Что это и зачем.

**Почему важно:** Где и как влияет на практику.

**Пример:**

```
...
```

**Связанные термины:** [[термин-1]], [[термин-2]]

---
_Составлено из сессии: <дата>_
```

---

For multiple related terms in one file:

```markdown
# Глоссарий: <тема>

---

## <Термин 1>

Определение. Пример.

---

## <Термин 2>

Определение. Пример.

---
_Составлено из сессии: <дата>_
```
