## Эквивалент _(optional)_

Для сложных синтаксических конструкций (comprehensions, декораторы, генераторы и т.п.) —
показывай развёртку в явный эквивалент.

```python
# короткая форма
[x * 2 for x in nums if x > 0]

# эквивалент через for
result = []
for x in nums:
    if x > 0:
        result.append(x * 2)
```
