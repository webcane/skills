**Эквивалент** _(optional — для сложных конструкций: comprehensions, декораторы, генераторы)_

```python
# короткая форма
[x * 2 for x in nums if x > 0]

# эквивалент через for
result = []
for x in nums:
    if x > 0:
        result.append(x * 2)
```
