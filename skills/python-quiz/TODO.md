## Улучшение качества вопросов

### Fact-grounding (снижение галлюцинаций)

При генерации вопроса в Init 2 добавлять поле `source_quote` — краткую цитату из .md,
на которой основан вопрос. Тогда некорректный ответ виден сразу при проверке в Init 2.5.

```json
{
  "id": 42,
  "type": "single",
  "topic": "generators",
  "subtopic": "yield vs return",
  "source_quote": "yield приостанавливает функцию и возвращает значение без завершения фрейма",
  "text": "Что происходит с локальными переменными функции после yield?",
  "options": ["A. ...", "B. ...", "C. ...", "D. ..."],
  "answer": "A"
}
```

В Init 2.5 добавить проверку: если `answer` противоречит `source_quote` — перегенерировать вопрос.
