# Разбор ошибок

К этому моменту у тебя есть список неверных ответов из Фазы 3.
Пройдись по ним по порядку номера вопроса.

## Формат каждой ошибки

Для каждого неверного ответа сгенерируй объяснение на месте и выведи блок:

```
❌ Вопрос N: [текст вопроса]
Твой ответ: [что ответил пользователь]
Правильный ответ: [верный ответ]
Объяснение: [2–4 предложения — почему именно так, в чём была ошибка]
```

## Навигация между ошибками

**Обрабатывай ошибки строго по одной.** Для каждой ошибки:

1. Выведи блок объяснения (формат выше).
2. **ОСТАНОВИСЬ.** Вызови инструмент `question` чтобы спросить пользователя:
   ```
   header: "Продолжить разбор"
   question: "Продолжить?"
   options: [
     {label: "Следующая ошибка", description: "Перейти к следующему неверному ответу"},
     {label: "Завершить разбор", description: "Закончить разбор ошибок"}
   ]
   multiSelect: false
   ```
3. **ЖДИ ответа пользователя.** Только после получения ответа:
   - Если «Следующая ошибка» — переходи к п.1 для следующей ошибки.
   - Если «Завершить разбор» — прекрати немедленно.

Если ошибок нет — скажи: «Отличная работа — все ответы верны!» и завершись.

Если разбор завершён досрочно или ошибки кончились: «Разбор завершён. Удачи на следующем тесте!»
