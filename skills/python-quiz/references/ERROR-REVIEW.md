# Разбор ошибок

К этому моменту у тебя есть список неверных ответов из Фазы 3.
Пройдись по ним по порядку номера вопроса.

## Формат каждой ошибки

Для каждого неверного ответа сгенерируй объяснение на месте и выведи блок:

```
❌ Вопрос N: [текст вопроса]
Твой ответ: [что ответил пользователь]
Правильный ответ: [верный ответ]
Объяснение: [2–4 предложения — почему именно так, в чём была ошибка]
```

## Навигация между ошибками

После каждой ошибки спрашивай:

```
header: "Продолжить разбор"
question: "Продолжить?"
options: ["Следующая ошибка", "Завершить разбор"]
```

Если ошибок нет — скажи: «Отличная работа — все ответы верны!» и завершись.

По окончании: «Разбор завершён. Удачи на следующем тесте!»
