# Python Quiz — Настройки теста

## Параметры

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `total_questions` | 20 | Вопросов в одном тесте (семплируется из банка) |
| `max_time_minutes` | 23 | Максимальное время теста. Превышение → тест не засчитан |
| `language` | ru | Язык вопросов и ответов (`ru` или `en`) |
| `difficulty_preset` | Базовый | Уровень сложности теста: `Базовый`, `Средний`, `Сложный`. Выбирается пользователем перед тестом. |
| `questions_per_topic` | 4 | Вопросов на каждую тему/подтему при `init` (диапазон 3–5) |
| `bank_file` | python-questions.json | Имя файла с банком вопросов |
| `results_log` | python-quiz-results-YYYY-MM-DD-HHMMSS.json | Имя файла с логом результатов (с датой и временем в имени) |

## Распределение типов вопросов

При `total_questions = 20`:

| Тип | Количество | % |
|-----|-----------|---|
| Single-choice | 8 | 40% |
| Multi-choice | 4 | 20% |
| Short answer | 7 | 35% |
| Coding task | 1 | 5% |

При другом `total_questions` масштабируй пропорционально, но coding task всегда 1 (последний).

Это же соотношение применяется при генерации вопросов на тему в режиме `init`.

## Как изменить настройки

Пользователь может указать настройки в запросе, например:
- «запусти тест на 10 вопросов»
- «тест на 30 минут»
- «сделай тест на английском»
- «базовый уровень» / «средний» / «сложный»

Если настройки указаны явно — они имеют приоритет над этим файлом.
