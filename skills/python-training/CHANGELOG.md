# Changelog — python-interview-coach

## [1.5.0] - 2026-07-11

### Changed
- Шаг 1 и Шаг 2 объединены: теперь вопросы задаются по одному с **немедленным разбором** после каждого ответа, сложность растёт от вопроса к вопросу
- Перенумерация шагов: 6 → 5 (Шаг 1 — вопрос-ответ-разбор, Шаг 2 — теория, Шаг 3 — уточняющие вопросы, Шаг 4 — задача, Шаг 5 — резюме)
- Структура теоретического разбора вынесена в `references/THEORY-STRUCTURE.md` — Шаг 2 ссылается на файл вместо инлайн-списка секций
- Все подтверждения и выборы переведены на встроенный инструмент `question` (универсально для VS Code и OpenCode): Шаг 1.7, Шаг 2 (после теории), Шаг 3 (после уточняющих), Шаг 4 (закрепить/продолжить), Шаг 5 (следующая тема/завершить)
- Версия: 1.4.0 → 1.5.0

## [1.3.0] - 2026-07-11

### Changed
- Шаг 1: вопросы теперь задаются по одному, а не скопом — после каждого ответа задаётся следующий
- Между шагами добавлено подтверждение перед переходом: пользователь явно подтверждает готовность к следующему шагу
- Перенумерация шагов: Шаг 5+6 → Шаг 5, Шаг 7 → Шаг 6
- Версия: 1.3.0 → 1.4.0

## [1.2.0] - 2026-07-09

### Changed
- Step 3: убран вывод в файл — теория показывается прямо в чате; переименован в «Теоретический разбор»
- Версия: 1.1.0 → 1.2.0

## [1.0.0] - 2026-07-08

### Added
- Initial release: интерактивный тренажёр Python-интервью уровня Middle+/Senior
- 22 темы от встроенных типов до dataclass и идиом Python
- Алгоритм из 7 шагов: вопросы → разбор → теория → задача → эталон → резюме
- Поведение интервьюера: наводящие вопросы, O-нотация, анализ сниппетов
- Режим итогового интервью на 40–50 вопросов
