# content-writer

Create authentic, engaging LinkedIn articles that share genuine professional insights and experiences.

## Overview

This skill guides you through planning and writing coherent, human-sounding LinkedIn content that follows the Minto Pyramid Principle, avoids buzzwords and clichés, and reads like genuine personal narrative rather than marketing copy.

## Key Features

- **Three-Mode Workflow**: Outline → Research/Verify (optional) → Write → Review Loop
- **Authentic Personal Narrative**: Focus on sharing genuine experiences and lessons learned
- **Readability Optimized**: Flesch-Kincaid 6-8th grade reading level
- **Collaborative Review**: Iterative refinement where you control what gets changed
- **Structured 6-Part Format**: Opening → Problem → Why conventional approach failed → Solution → Lesson → Discussion invitation

## Modes

### 1. OUTLINE MODE
Plan your article structure before writing. Get word count targets and story beats for each section.

### 2. RESEARCH/VERIFY MODE (Optional)
Verify facts, statistics, and claims. Skip by default if your article is pure personal narrative.

### 3. WRITE MODE
Draft the complete article following the outline and style guidelines.

### 4. REVIEW & REVISION LOOP
Collaborate on refinements. Request rephrasing of specific sections until satisfied.

## Style Highlights

- **First-person authentic voice** — write like you speak to colleagues
- **Sentence rhythm variation** — mix short (5-10 words), medium (15-20), long (25+) for engagement
- **No corporate jargon** — 40+ specific words/phrases to avoid
- **LinkedIn-optimized** — image recommendations, title formula, algorithm insights

## What It Helps With

✅ Sharing lessons learned from professional challenges  
✅ Creating thought leadership that feels genuine  
✅ Documenting discoveries and solutions you've experienced  
✅ Writing content that avoids buzzwords and clichés  
✅ Structuring personal narratives for impact  

## Target Audience

LinkedIn professionals who want to share authentic insights without sounding corporate or promotional.

## Installation

1. Copy the `content-writer` directory to your skills folder
2. The skill will be available in Claude with the ability to create LinkedIn articles

## Usage

**Start outlining an article:**
```
Help me outline a LinkedIn article about [topic]
```

**Add research for data-driven content:**
```
Can you verify the stats and integrate current trends?
```

**Write the article:**
```
Write the article based on this outline
```

**Refine specific parts:**
```
Make the opening more conversational
Tighten up the problem section
This feels too promotional—adjust the tone
```

**Finish up:**
```
Done!
```
