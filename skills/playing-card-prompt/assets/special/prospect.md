# Special Card Type: Prospect

A single special card that depicts all 12 court figures (King/Queen/Jack × 4 suits) arranged together in one image — a portrait sheet or reference card for the full court. Each figure is a named historical, cultural, or thematic person. The wizard collects all 12 names before generating ONE prompt.

## Special type line
```
court figure prospect card, all 12 named figures arranged in a grid, 4 suits × 3 ranks, each figure a distinct named portrait,
```
