# Pattern: Japanese (ukiyo-e)

Edo-period ukiyo-e woodblock print aesthetic: bold outlines, flat color fields, and
traditional motifs (waves, cherry blossoms, cloud bands). Each section below is a
layer fragment; copy lines verbatim (trailing comma kept) into the matching layer slot
during assembly. See "Layers and `[STYLE_BLOCK]` assembly" in `references/REFERENCE.md`
for how these combine per card type.

## Background
```
warm cream washi paper texture, subtle fibrous grain,
```

## Decor (background pattern)
```
indigo blue, vermilion red, and mustard ochre flat color fields,
```

## Ornaments
```
stylized wave, cherry blossom, and cloud-band motifs along the borders,
```

## Highlights / Overlays
```
fine mica-dust shimmer along key outlines,
```

## Center motif style
```
bold confident black outlines, minimal interior shading, flat unmodulated color fields,
traditional ukiyo-e woodblock print illustration,
```

## Figure detail
```
pale ivory skin tones with minimal shading,
```

## Face Style
```
stylized ukiyo-e facial features with simplified, composed and reserved expression,
```

## Finish
```
matte handmade-paper finish,
visible woodblock print texture,
highly detailed Japanese ukiyo-e-style playing card,
```
