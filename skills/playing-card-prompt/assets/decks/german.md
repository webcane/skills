# Deck: German

German-suited deck. No Unicode glyphs — SUIT_SYMBOL is a described shape.

## Ranks available
Default standard set: A, 2–10, J, Q, K, Joker (optional).
(Authentic German packs use 6–10, Unter, Ober, König, Daus and have no Queen —
to model that, edit this rank list and the lettering. Left standard by default.)

## Default lettering system
German / Austrian (K / D / B).
