# Deck: Swiss (Swiss-German)

Swiss-suited deck. SUIT_SYMBOL is a described shape.

## Ranks available
Default standard set: A, 2–10, J, Q, K, Joker (optional).
(Authentic Swiss packs use 6–10, Under, Ober, König, As and have no Queen — edit
this list to model that. Left standard by default.)

## Default lettering system
German / Austrian (K / D / B).
